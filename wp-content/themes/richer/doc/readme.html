Theme Installation

To install this theme you must have a working version of WordPress already installed. For information in regard to installing the WordPress platform, please see the WordPress Codex - http://codex.wordpress.org/Installing_WordPress

Theme installation

After you finish setting up your WordPress installation, you have to install this theme.

You can do it in two ways:

    1. 1. FTP Upload:
      Unzip the “richer.zip” file and upload the contents into the /wp-content/themes folder on your server.

    2. 2. Wordpress Upload:
      Navigate to Appearance > Add New Themes > Upload. Go to browser, and select the zipped theme folder. Hit “Install Now” and the theme will be uploaded and installed

Theme activation

Once you have uploaded the theme, activate your theme in Dashboard > Appearance > Themes. Using Themes http://codex.wordpress.org/Using_Themes

If you are experiencing any issues during the installation I would recommend to check out this Screencast by Envato

Plugins installation

Now, at this stage you have noticed a warning, asking you to install recommended plugins. This is needed to get Contact Forms, Revolution slider. You just need to click begin installing plugins and you will be redirected to the plugin installation page where you can install any plugins what you need.

Importing demo content

We have included option in dashboard,this allows you to quickly set up your theme the same way our demo is set up.

If you need not to use Import button in theme options. You can import content with wordpress importer plugin. Instructions on how to do this are below.

  1. 1. Log into your WordPress backend
  2. 2. Navigate to Tools > Import
  3. 3. Click on "WordPress". If you haven't imported anything yet, you will need to install "Importer" plugin.
  4. 4. Select the .xml file in the "data import" folder in the zip-file you've downloaded from Themeforest.
  5. 5. Now make sure to Select the Menu in Appearance > Menus and set up "Home" as front page and "Blog" as posts page under Settings > Reading.

Be aware that downloading Dummy Content may take a while, depending on the speed of your internet connection!
Regarding built in importer. If you get white screen after importing, please refresh page and wait a minute, you should be redirected to Theme options -> General!

Revolution slider setup

Theme Options Setup

If you want to change the general theme options, go to your Dashboard > Appearance > Theme Options. Here you can change a lot of options of your new theme (fonts, colors, background, layouts etc.)

Notice: Before adjusting the different settings, you should first click on the Options Reset button, to insert all default data to the database and make sure everything looks good.

Blog post formats

Theme supports a lot of post formats such as: Standard, Image, Gallery, Link, Quote, Audio, Video.
If you need to show post thumbnail, you need to set featured image.

Standard

When writing or editing a Post, Standard is used to designate that no Post Format is specified. Also if a format is specified that is invalid then standard (no format) will be used.

Image

This post format allows you to show a single image. You need to set your featured image.

Gallery

This post format helps you to display a gallery of images. When you use gallery post format you have post gallery options below your content editor. You can upload images for you gallery and specify layout preview.

Link

Use this post format to attach post title with link to another site. You can use "Post link" area below content editor.

Quote

If you need to display quote, please, use this format. Insert in content editor quote text and use options below editor to attach author and link.

Audio

This post format helps you to display an audio track from soundcloud. Use "Post audio" area below content editor to insert embed code from soundcloud.

Video

This post format helps you to display a video from youtube, vimeo, etc. Use "Post video" area below content editor to insert video ID or own embed code.

Custom Sidebars

There are 3 widget areas that comes with this theme - Blog Widgets, Footer Widgets and Top Area Widgets, where you can drop your Widgets. You can also define custom sidebars for each page.

To create Custom Sidebars, follow these steps:

  • 1. Navigate to Appearance > Sidebars.
  • 2. Click Add New Sidebar, choose a Name and click OK to save it.
  • 3. Now you can add Widgets to your Sidebar under Appearance > Widgets.
  • 4. Once you've added your Widgets you can select your Sidebar when creating a new page at the bottom of the Page. Just select your sidebar and choose the "Default" page template or "Page: Sidebar Left" page template from the "Page Attributes".

Shortcodes

Shortcode is a powerful content building tool http://codex.wordpress.org/Shortcode.

Theme comes packed with a lot of shortcodes allowing you to add complex styled content with little effort. The shortcodes are available under the visual formatting options on the post / page editor. Make sure you are on the "visual editor" tab, the shortcode buttons are on the second row. Simply hover over a button to see which shortcode it is and then click it and the shortcode will be automatically added, then you need to populate the content and parameters.

Examples of the shortcodes you can find here http://richer.artstudioworks.net/shortcodes/

Translation

This theme is localized, which means it can easily be translated in your own language(easy but quite time consuming). To translate it you need to follow these steps:
  • 1. Download and install http://www.poedit.net/
  • 2. Go to /wp-content/themes/theme_name/framework/languages/ and open en_EN.po file.
  • 3. In the window that appears you have all the strings that were used in this theme, so you just need to go through them and write a translation.
  • 4. When you’re done go to File > Save as and save your translation in a *.po file.
  • 5. You should name your file pt_LANG. It’s just a naming convension WordPress_in_Your_Language.
  • 6. When you’re done translating open wp-config.php from your WordPress root folder and replace this line:
    • define ('WPLANG', ''); with define ('WPLANG', 'pt_LANG');
